# Django Assignment Answers

## Question 1

### manage.py
This file provides command-line tools for managing the Django project, such as running the server, creating migrations, and executing administrative tasks.

### settings.py
This file contains the project's configuration settings, including installed apps, database setup, security options, and other global settings.

### urls.py
This file defines the URL routes for the project and maps each URL to the appropriate view or application.
